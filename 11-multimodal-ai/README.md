# Multimodal AI Learning Resources

## Vision-Language Models

- [Hugging Face: Vision Language Models Explained](https://huggingface.co/blog/vlms) - VLM 权威入门，从 Hugging Face 官方博客，覆盖图像与文本联合训练原理。 `Article` `EN`
- [Best Multimodal Vision Models 2025 - Koyeb](https://www.koyeb.com/blog/best-multimodal-vision-models-in-2025) - Gemma 3、Qwen 2.5 VL、Pixtral、Phi-4 Multimodal、DeepSeek Janus Pro 实用模型选型指南。 `Article` `EN`
- [Google DeepMind: Gemma 3 Technical Report](https://ai.google.dev/gemma) - Google 开源多模态模型家族，支持图像/视频/文本输入。 `Docs` `EN`
- [Qwen-VL Series (Alibaba)](https://huggingface.co/Qwen) - 阿里巴巴通义千问视觉语言模型系列，中文多模态能力领先。 `Model` `ZH/EN`
- [LLaVA: Large Language-and-Vision Assistant](https://github.com/haotian-liu/LLaVA) - 开源视觉语言模型，基于 ViT + LLaMA 架构，20k+ stars。 `Repo` `EN`

## Image Generation

- [ComfyUI](https://github.com/comfy-org/comfyUI) - 70k+ stars，最强大的模块化扩散模型 GUI/后端，节点式工作流。 `Repo` `EN`
- [Stable Diffusion WebUI (Automatic1111)](https://github.com/AUTOMATIC1111/stable-diffusion-webui) - 150k+ stars，最流行的 Stable Diffusion Web 界面。 `Repo` `EN`
- [FLUX by Black Forest Labs](https://github.com/black-forest-labs/flux) - Stable Diffusion 原班人马打造的新一代图像生成模型。 `Repo` `EN`
- [Midjourney Official Docs](https://docs.midjourney.com/) - 商业图像生成领域的领导者。 `Docs` `EN`
- [DALL-E 3 - OpenAI](https://openai.com/dall-e-3) - OpenAI 官方图像生成模型，集成在 ChatGPT 中。 `Docs` `EN`

## Video Generation

- [Kling AI (Kuaishou)](https://klingai.com/) - 快手可灵 AI，中国视频生成领域领先者。 `Tool` `ZH/EN`
- [Luma Dream Machine](https://lumalabs.ai/dream-machine) - 高质量视频生成，支持图片/文字转视频。 `Tool` `EN`
- [Runway Gen-3 Alpha](https://runwayml.com/) - 视频生成行业先驱，Gen-3 Alpha 支持精细控制。 `Tool` `EN`
- [Video Generation Tools Comparison 2026](https://pollo.ai/hub/kling-ai-vs-runway-vs-luma-ai-vs-pollo-ai) - Kling vs Runway vs Luma vs Pollo 全面对比。 `Article` `EN`
- [Google Veo](https://deepmind.google/technologies/veo/) - Google DeepMind 视频生成模型，1080p 高质量输出。 `Docs` `EN`

## Multimodal RAG

- [Building Multimodal Search and RAG - DeepLearning.AI](https://learn.deeplearning.ai/courses/building-multimodal-search-and-rag/information) - Andrew Ng 平台免费课程，构建多模态搜索和 RAG 系统。 `Free Course` `EN`
- [Multimodal RAG with LangChain](https://blog.langchain.dev/multimodal-rag-with-langchain/) - LangChain 官方多模态 RAG 实现指南，支持图像+文本检索。 `Article` `EN`
- [ColPali: Visual Document Retrieval](https://github.com/illuin-tech/colpali) - 用视觉模型替代传统 OCR+嵌入的文档检索方法，效率显著提升。 `Repo` `EN`

## Audio & Speech

- [OpenAI Whisper](https://github.com/openai/whisper) - 80k+ stars，最强大的开源语音识别模型，支持 99 种语言。 `Repo` `EN`
- [ElevenLabs](https://elevenlabs.io/) - AI 语音合成领域领导者，支持语音克隆和多语言 TTS。 `Tool` `EN`
- [Google NotebookLM](https://notebooklm.google.com/) - Google AI 笔记工具，支持音频概览（AI 播客生成）。 `Tool` `EN`

## Learning Path

| Stage | Resource | Focus |
|-------|----------|-------|
| VLM Basics | HF VLM Blog + LLaVA | 视觉语言模型原理 |
| Image Gen | ComfyUI + Stable Diffusion WebUI | 图像生成实践 |
| Video Gen | Kling/Luma/Runway 对比 | 视频生成工具选型 |
| Multimodal RAG | DeepLearning.AI 课程 + LangChain 指南 | 多模态检索增强 |
| Audio | Whisper + ElevenLabs | 语音识别与合成 |
