# Awesome AI Learning Resources

A curated collection of high-quality learning resources for AI, covering Prompt Engineering, AI Agents, MCP, Skills, Claude Code, and AGI.

> Continuously updated. Contributions welcome.

## Categories

| Topic | Description |
|-------|-------------|
| [Prompt Engineering](./prompt-engineering/) | Tutorials, courses, and best practices for prompt engineering |
| [AI Agent](./ai-agent/) | Agent frameworks, courses, and architecture patterns |
| [MCP (Model Context Protocol)](./mcp/) | MCP protocol documentation, tutorials, and server ecosystem |
| [Skills / Function Calling](./skills/) | Tool use, function calling, and agent skills development |
| [Claude Code](./claude-code/) | Claude Code skills, hooks, plugins, and MCP integration |
| [AGI](./agi/) | AGI research, timelines, and policy reports |

## Recommended Learning Path

```
Beginner
  ├── Prompt Engineering (Andrew Ng course)
  ├── AI Agent Basics (Hugging Face / DeepLearning.AI)
  └── Official docs (OpenAI / Anthropic / Google)
Intermediate
  ├── Agent Frameworks (LangGraph / CrewAI / AutoGen)
  ├── MCP Protocol & Server Development
  └── Skills & Function Calling
Advanced
  ├── Multi-Agent Architecture Patterns
  ├── Production Deployment & Observability
  └── AGI Research & Safety
```
