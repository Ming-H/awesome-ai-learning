# Claude Code Learning Resources

## Official Documentation

- [Claude Code Hooks Reference](https://code.claude.com/docs/en/hooks) - Hooks 是用户定义的 shell 命令/HTTP 端点/LLM prompts，在生命周期事件中触发。 `Docs` `EN`
- [Claude Code Hooks Guide](https://code.claude.com/docs/en/hooks-guide) - 使用 hooks 进行文件格式化、阻止命令、发送通知和工作流自动化。 `Docs` `EN`
- [Claude Code Features Overview (Plugins)](https://code.claude.com/docs/en/features-overview) - 插件将 skills、hooks、subagents 和 MCP 服务器打包为可安装单元。 `Docs` `EN`
- [Agent Skills Overview](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview) - Agent Skills 如何打包说明、元数据和资源来扩展 Claude 功能。 `Docs` `EN`
- [Claude Agent SDK Overview](https://code.claude.com/docs/en/agent-sdk/overview) - 提供与 Claude Code 相同工具链的 SDK，支持 Python 和 TypeScript。 `Docs` `EN`
