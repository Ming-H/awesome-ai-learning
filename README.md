```
  ├── AI Agent 基础 / Basics (Hugging Face / DeepLearning.AI)

高级 / Advanced
  ├── 多 Agent 架构模式 / Multi-Agent Architecture Patterns
  ├── AI 安全与对齐 / Safety & Alignment (RLHF, Red Teaming)
  ├── 推理优化与部署 / Inference Optimization & Deployment
  └── AGI 研究与治理 / AGI Research & Governance
```
