- [ChatGPT Prompt Engineering for Developers](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers) - Andrew Ng + OpenAI 最推荐的入门课程，免费 1-2 小时。 `Free Course` `EN`
- [AI Prompting for Everyone](https://www.deeplearning.ai/courses/ai-prompting-for-everyone) - Andrew Ng 面向非开发者的提示工程课程。 `Free Course` `EN`
